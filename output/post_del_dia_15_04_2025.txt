1️⃣ Uno. **Paquete Malicioso de PyPI Ataca la API de MEXC para Robar Credenciales.** Investigadores de ciberseguridad han descubierto un paquete malicioso en PyPI diseñado para redirigir órdenes de trading en la plataforma MEXC y robar tokens. Aunque ya no está disponible, se descargó más de 1,000 veces.

2️⃣ Dos. **Vulnerabilidad Crítica en Apache Roller Permite Persistencia de Sesiones No Autorizadas.** Se ha revelado una grave vulnerabilidad en el software de blogging Apache Roller que permite a actores maliciosos mantener acceso no autorizado a cuentas incluso después de un cambio de contraseña, afectando todas las versiones hasta 6.1.4.

3️⃣ Tres. **La Mayoría de las Extensiones de Navegador Pueden Acceder a Datos Sensibles de Empresas.** Un informe revela que el 53% de las extensiones utilizadas en entornos empresariales tienen permisos que les permiten acceder a datos críticos, lo que representa un riesgo significativo para la seguridad organizacional.

4️⃣ Cuatro. **Desarrolladores de Criptomonedas Objetivo de Malware de Python Disfrazado como Retos de Programación.** Un grupo de amenazas vinculado a Corea del Norte ha sido identificado entregando malware a desarrolladores a través de ofertas de trabajo falsas en LinkedIn, utilizando un malware llamado RN Stealer para robar información sensible.

5️⃣ Cinco. **Piratas Informáticos Chinos Apuntan a Sistemas Linux Usando Malware SNOWLIGHT.** Un nuevo ataque atribuido al grupo UNC5174 utiliza una variante de malware denominada SNOWLIGHT para infectar sistemas Linux, destacando el uso de herramientas de código abierto para ocultar sus actividades.
