1️⃣ Uno. **ASUS Confirma Vulnerabilidad Crítica en Routers AiCloud; Usuarios Urgidos a Actualizar Firmware.** ASUS ha revelado una falla de seguridad que permite a atacantes remotos ejecutar funciones no autorizadas en ciertos routers AiCloud. 2️⃣ Dos. **Bloqueos Masivos de Cuentas en Microsoft Entra Debido a Nueva Función de Seguridad.** Administradores de Windows reportan bloqueos de cuentas provocados por falsos positivos en la nueva aplicación de detección de credenciales filtradas de Microsoft Entra. Se investiga el origen del problema.

3️⃣ Tres. **Nuevo Malware en Android Roba Tarjetas de Crédito para Ataques NFC.** Un nuevo malware llamado 'SuperCard X' ataca dispositivos Android usando técnicas de relay NFC para realizar transacciones no autorizadas con datos de tarjetas comprometidas.

4️⃣ Cuatro. **Exploits Públicos Disponibles para Vulnerabilidad Crítica de Erlang/OTP SSH; Parches Necesarios.** Investigadores han publicado exploits para una vulnerabilidad crítica en Erlang/OTP que permite a atacantes ejecutar código de forma remota. Se recomienda actualizar los dispositivos afectados de inmediato.

5️⃣ Cinco. **Paquetes Maliciosos en npm Simulan API de Bot de Telegram para Instalar Backdoors.** Investigadores de ciberseguridad han encontrado paquetes en el registro de npm que imitan una popular biblioteca de Telegram, facilitando la instalación de backdoors SSH en sistemas Linux.
